# 대학생 프로젝트 매칭, 협업 플랫폼
 주제 기획서 — 대학생 프로젝트 매칭, 협업 플랫폼(다모여)
## 주제
 대학, 학과 기반으로 팀원을 매칭하고, 지원–면접–승인을 거쳐 모집 완료 후 업무·일정·회의·결과물을 관리하며, 팀장 권 한 자동 승계와 멘토 검토까지 지원하는 대학생 프로젝트 매칭·협업 플랫폼
## 목적
 대학생은 공모전, 학과 프로젝트, 교양 팀플, 교내 활동 등 다양한 협업 프로젝트를 수행하지만, 팀원을 구하는 채널이 학과 단톡방, 지인 소개 등으로 분산되어 있어 적합한 팀원을 찾기 어렵습니다. 팀이 구성된 이후에도 업무, 일정, 결과물이 여러 도구로 흩어져 진행 상황을 체계적으로 파악하기 힘든 문제가 있습니다. 기존 협업 도구는 이미 구성된 팀의 업무 관리에 집중되어 있어 팀원 모집, 매칭 단계부터 지원하지는 못합니다. 대학, 학과라는 명확한 울타리 안에서 신뢰할 수 있는 팀원을 매칭하고, 모집부터 완료까지 하나의 흐름으로 잇는 서비스가 필요합니다. 본 프로젝트는 대학, 학과 기반 매칭으로 팀원을 모집하고, 지원–면접–승인을 거쳐 팀을 구성하며, 이후 업무, 일정, 회의, 결과물 관리와 멘토 검토까지 하나의 플랫폼에서 제공하는 대학생 프로젝트 협업 서비스를 개발하고자 합니다.
## 기대효과
 팀원 매칭 효율화 — 대학, 학과, 관심 분야, 태그 조건 매칭으로 적합한 팀원을 빠르게 찾아 팀 구성 비용을 줄임.<br>
 모집–완료 과정의 일원화 — 모집, 지원, 업무, 결과물을 한 곳에서 관리하여 흩어져 있던 협업 채널을 통합. <br>
 체계적인 진행 관리 — 업무 상태, 진행도, 역할 분담을 시각적으로 파악해 진행 지연과 무임승차를 방지. <br>
 권한 기반의 신뢰성 확보 — 팀장, 팀원, 멘토 권한 분리와 학교 인증으로 신뢰도 높은 협업 환경 제공. <br>
 실무형 개발 경험 축적 — 상태머신, 트랜잭션, 권한 제어, 통계 등 백엔드 핵심 기술을 실제 서비스에 적용. <br>
